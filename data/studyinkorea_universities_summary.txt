Data Structure:
--------------
Each university entry contains:
- univCd: University code (e.g., "100249")
- nameKo: Korean name (e.g., "세명대학교")
- location: Korean region name (e.g., "충청북도", "서울")
- type: University type (e.g., "사립", "국립", "공립")
- description: University description (optional)
- degrees: Array of degree levels offered
